# 🌐 3D Animated Website

An immersive **3D animated web experience** built using modern front-end technologies.
This project focuses on delivering smooth animations, interactive UI, and visually engaging transitions to enhance user experience.

---

## 🚀 Project Overview

This website is designed as a **3D interactive landing page**, showcasing modern web animation techniques. It demonstrates how motion, depth, and smooth transitions can improve UI/UX in real-world web applications.

---

## 🛠️ Tech Stack

* **HTML5** – Structure and layout
* **CSS3** – Styling and responsiveness
* **JavaScript (ES6)** – Logic and interactivity
* **GSAP (GreenSock Animation Platform)** – Advanced animations

---

## ✨ Key Features

* 🎬 Smooth 3D-like animations and transitions
* 🌀 Scroll-based animation effects
* ⚡ High-performance animation using GSAP
* 🖱️ Interactive elements (hover & motion effects)
* 📱 Fully responsive design
* 🎨 Modern UI with clean layout

---

## 🧠 Animations & Effects

* GSAP timeline animations
* Scroll-triggered animations
* Transform-based 3D effects
* Smooth transitions between sections
